<h1>Page Loader - Hexlet project No. 3</h1>

Downloads web pages to specified directory

<h1>Install:</h1>

npm install -g sergye-page-loader

<h1>Usage example:</h1>

page-loader --output /var/tmp https://hexlet.io/courses
